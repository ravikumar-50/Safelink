# 🛡️ SafeLink AI: Advanced Cybersecurity Protection

SafeLink AI is a state-of-the-art cybersecurity platform designed to protect users from the evolving landscape of phishing attacks and malicious communications. By combining advanced artificial intelligence with real-time threat intelligence, SafeLink AI provides an essential layer of security for everyday digital interactions.

## ✨ Key Features

- **🤖 AI-Driven Intent Analysis**: Uses cutting-edge AI to analyze the context and intent of messages, identifying subtle social engineering cues.
- **🔍 Multi-Layered URL Scanning**: Integrates with industry-standard threat intelligence providers (VirusTotal, Google Safe Browsing).
- **🌐 Deep Domain Intelligence**: Pulls real-time data on domain age, registrar legitimacy, and server geographical location.
- **📊 Intuitive Risk Scoring**: Categorizes threats into SAFE, SUSPICIOUS, or DANGEROUS with detailed reasoning.
- **🔐 Secure User Dashboard**: History of all scans is securely stored for registered users.
- **🚀 Ultra-Modern Interface**: A high-performance, responsive UI built with a "cyberpunk" aesthetic.

## 🛠️ Technical Stack

- **Next.js 16** (App Router)
- **Tailwind CSS & Framer Motion**
- **MongoDB & Mongoose**
- **NextAuth.js**
- **Groq AI Engine**

## 🚀 Getting Started

First, install dependencies:

```bash
npm install
```

Set up your environment variables in `.env.local`:

```
MONGODB_URI=your_mongodb_uri
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000
GROQ_API_KEY=your_groq_api_key
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
